#####################
Project Description##
#####################
	The Ultimate Puzzle Game contains 3 modes. 'Customize Your Own Puzzle' allows user to set the number of rows and columns of the puzzle with an image of their choice. 'Color Puzzle' mode randomly generates color gradient and levels. 'Music Puzzle' is a puzzle in motion. Users can upload a .wav or .mp3 file. Find more detailed descriptions in the Instruction & Help page in the game.

############
How to run##
############
	First, please install the following modules and libraries: 
		pyaudio; 
		spottily; 
		spotify.util; 
		matplotlib; 
		numpy; 
		tkinter; 
		PIL; 
		pydub; 
		requests;
		sys;
		os;
		csv;
		time;
		random;
		wave;
		math.

	Please make sure FakeButton.py, saveProgress.py, spotifyInfo.py, beatDetection.py, HelperWidgets.py, grapher.py, and cmu_112_graphics.py are in the same folder with theUltimatePuzzleGame.py. And run theUltimatePuzzleGame.py to access the game.

	For Spotify authorization, please copy and paste the link you'll be redirected back to console in order to be authorized. This can take a bit long depending on the number of playlists that you have in your Spotify account. Some Spotify accounts does not have a username, and there is not a way to use your Spotify account if you're one of accounts, sorry :)
